AlphaSync Help

AlphaSync is an alternative means to get file data on or off an AlphaSmart Neo or compatible device.

AlphaSync provides a Finder-like interface to browse files stored on a Neo, with drag-and-drop to allow new files to be quickly copied to or from the device. It also provides a quick backup feature that will automatically copy files from the Neo to a predefined folder on your Mac.

AlphaSync is not a replacement for AlphaSmart Manager. In particular, it will not make changes to your applets or software on the Neo, nor is it intended to work with large numbers of devices connected to the same computer.

For more information on the Neo and AlphaSmart Manager, visit the AlphaSmart website.
