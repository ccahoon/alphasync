Quick Backup



To backup all files for all currently connected devices, click the green backup icon in the toolbar or press Command-B. All files that are accessible in the main window will be backed up, even if they are not selected.

Backups are written to a fixed place on disk - by default the folder named AlphaSync in your Documents directory. The backup location can be changed in the preference settings.

The current backup location can be opened in the finder from the AlphaSync File menu or by pressing Command-R
