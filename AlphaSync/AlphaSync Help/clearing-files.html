Clearing Files



Clearing a single file



To clear the contents of a single file, select the file in the main window and click the blue 'Clear' toolbar icon. The contents of the file are erased.


Clearing all files for an applet



To clear all files for an applet, select the applet in the main window and click the red 'Clear' toolbar icon.
